# Aulirya

L'agenda intelligent pour les enseignants. Aulirya aide a planifier les cours,
suivre les programmes et organiser les classes avec une interface claire et
apaisante.

## Tech stack

**Frontend**

-   React + TypeScript
-   TanStack Router + TanStack Query
-   Tailwind CSS
-   Clerk (auth)

**Backend**

-   Elysia (Bun)
-   oRPC
-   SurrealDB

**Infra**

-   Docker / Docker Compose

## Quick summary

Application fullstack pour enseignants : planning hebdomadaire, gestion des
classes, suivi des programmes et progression des lecons.
